###### 树
怎么组织项目才能够既容易又灵活？“树”是我的一种尝试，按照打开的历史来形成项目。

一般是以目录为封闭单元来构成项目，这种项目需要配置，而一旦配置固定后，跨项目、项目交叉中出现的文件便成为麻烦，而项目里包含另外项目有时也会冲突。
